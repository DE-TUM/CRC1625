# CRC 1625 Ontology
This folder contains the following ontology files:
- `crc.ttl`: Our CRC 1625 Ontology
- `pmd_core.ttl`: PMD-core ontology 2.0.8, which is imported by our ontology.
- `oce.ttl` and `oce.owl`: Ontology of Chemical Elements (OCE), used to link to OCE/CheBI elements in Chemical Compositions.
- `ontology.drawio`: Overview of the structure of the Knowledge Graph (both `ABox` and `TBox`). **Note**: This is best viewed in **light mode**.
